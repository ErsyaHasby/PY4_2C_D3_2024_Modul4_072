# Panduan Menambahkan Gambar Onboarding

## Cara Menambahkan Gambar:

1. **Siapkan 3 gambar** dengan nama:
   - `onboarding1.png` (atau .jpg)
   - `onboarding2.png` (atau .jpg)
   - `onboarding3.png` (atau .jpg)

2. **Copy gambar** ke folder ini: `assets/images/`

3. **Rekomendasi ukuran gambar:**
   - Resolusi: 512x512 px atau 1024x1024 px
   - Format: PNG (untuk transparansi) atau JPG
   - Ukuran file: < 500KB per gambar

## Sumber Gambar Gratis:

- **Undraw**: https://undraw.co/illustrations (ilustrasi SVG/PNG gratis)
- **Storyset**: https://storyset.com/ (ilustrasi gratis)
- **Flaticon**: https://www.flaticon.com/ (icon dan ilustrasi)
- **Freepik**: https://www.freepik.com/ (berbagai ilustrasi)

## Tema yang Disarankan:

- **onboarding1.png**: Ilustrasi "Welcome" atau "Book"
- **onboarding2.png**: Ilustrasi "Counter" atau "Task Management"
- **onboarding3.png**: Ilustrasi "Login" atau "Getting Started"

## Contoh Pencarian:

Di Undraw, cari keyword:
- "welcome"
- "counter"
- "login"
- "task"
- "checklist"

Setelah download, rename menjadi `onboarding1.png`, `onboarding2.png`, `onboarding3.png`
